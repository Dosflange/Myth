This page describes the following instruction:

  • Group: FIX
  • Mnemonic: M2
  • Opcode: 14d, 0Eh
  • Encoding: 00001_110b (FIX_ADDEND)
  • Pseudo-code: r-=2; pc++

This instruction subtracts 2 to the Result register.
