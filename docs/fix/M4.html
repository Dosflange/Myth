This page describes the following instruction:

  • Group: FIX
  • Mnemonic: M4
  • Opcode: 12d, 0Ch
  • Encoding: 00001_100b (FIX_ADDEND)
  • Pseudo-code: r-=4; pc++

This instruction subracts 4 from the Result register.
