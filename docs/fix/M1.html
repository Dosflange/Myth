This page describes the following instruction:

  • Group: FIX
  • Mnemonic: M1
  • Opcode: 15d, 0Fh
  • Encoding: 00001_111b (FIX_ADDEND)
  • Pseudo-code: r-=1; pc++

This instruction subtracts 1 to the Result register.
