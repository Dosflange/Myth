This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: r3
  • Opcode: 107d, 6Bh
  • Encoding: 01_10_1_011b (DIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: L3 = r; pc++

Puts R into local memory at offset 0xFB.
