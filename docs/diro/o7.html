This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: o7
  • Opcode: 127d, 7Fh
  • Encoding: 01_11_1_111b (TRAP_REG_DIRECTION_INDEX)
  • Pseudo-code: L7 = o; pc++

Puts O into local memory at offset 0xFF.
