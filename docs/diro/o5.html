This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: o5
  • Opcode: 125d, 7Dh
  • Encoding: 01_11_1_101b (DIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: L5 = o; pc++

Puts O into local memory at offset 0xFD.
