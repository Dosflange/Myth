This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: o4
  • Opcode: 124d, 7Ch
  • Encoding: 01_11_1_100b (TRAP_REG_DIRECTION_INDEX)
  • Pseudo-code: L4 = o; pc++

Puts O into local memory at offset 0xFC.
