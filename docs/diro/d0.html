This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: d0
  • Opcode: 72d, 48h
  • Encoding: 01_00_1_000b (DIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: L0 = d; pc++

Puts D into local memory at offset 0xF8.
