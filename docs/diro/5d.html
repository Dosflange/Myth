This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: 5d
  • Opcode: 69d, 45h
  • Encoding: 01_00_0_101b (DIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: d=L5; pc++

Gets D from local memory at offset 0xFD.
