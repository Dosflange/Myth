  • Group: DIRO
  • Mnemonic: 4r
  • Opcode: 100d, 64h
  • Encoding: 01_10_0_100b (TRAP_REG_DIRECTION_INDEX)
  • Pseudo-code: r=L4; pc++

Gets R from local memory at offset 0xFC.
