This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: ib
  • Opcode: 218d, DAh
  • Encoding: 1_101_1010b (PAIR_SRC_DST)
  • Pseudo-code: pc+=i
  • I:INNER B:BRANCH

  • Source is the I register.
  • Add source to the Program Counter.

The 'B' destination takes the source value as the offset relative to the current value of the Program Counter. Writing to 'B' triggers an unconditional jump to the address offset that is the sum of the Program Counter and the source byte within the current code page. See In-Page Branching for more on this.
