This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: do
  • Opcode: 176d, B0h
  • Encoding: 1_011_0000b (TRAP_SRC_DST)
  • Pseudo-code: o=d; pc++
  • D:DATA O:OPERAND

  • Source is the D register.
  • Destination is the memory cell at page D offset O.

The 'D' source is the page index used for memory operations via the 'M' source or destination.
