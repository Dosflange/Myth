This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: nt
  • Opcode: 141d, 8Dh
  • Encoding: 1_000_1101b (TRAP_SRC_DST)
  • Pseudo-code: if r!=0 then pc=ram[c][pc] else pc++
  • N:NUMBER T:TRUE

  • Source is the literal following the current opcode.
  • If R is NOT zero, set the Program Counter to source.

The 'N' source is a literal byte embedded in the instruction stream. Reading 'N' acts in the same way as an instruction fetch, only it fetches a literal. The 'T' destination takes the source value as the target offset of a conditional jump (R!=0) to within the current code page. See In-Page Branching for more on this.
