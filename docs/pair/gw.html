This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: gw
  • Opcode: 188d, BCh
  • Encoding: 1_011_1100b (TRAP_SRC_DST)
  • Pseudo-code: if i!=0 then pc=g else pc++; i--
  • G:GLOBAL W:WHILE

  • Source is the G register.
  • If I is NOT zero, set the Program Counter to source. Then in either case decrement I.

The 'G' source is the page index used for memory operations via the 'M' source or destination. The 'W' destination takes the source value as the target offset of a conditional jump (I != 0) to within the current code page. See In-Page Branching for more on this.
