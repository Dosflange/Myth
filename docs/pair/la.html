This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: la
  • Opcode: 169d, A9h
  • Encoding: 1_010_1001b (TRAP_SRC_DST)
  • Pseudo-code: if (o+ram[l][o])>255 then d+=1; pc++
  • L:LOCAL A:ADD

  • Source is the memory cell at page L offset O.
  • Add source to O. If this produces a carry bit, add the carry bit to D.

The 'L' source is the value of a memory cell in the L-page. See Page Registers for more information on the three page register (D, L, C). The 'A' destination takes the source value as the addend which is added to the Data pointer (register pair D:O). See In-Page Branching for more on this.
