This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: gm
  • Opcode: 177d, B1h
  • Encoding: 1_011_0001b (PAIR_SRC_DST)
  • Pseudo-code: ram[g][o]=g; pc++
  • G:GLOBAL M:MEMORY

  • Source is the G register.
  • Destination is the memory cell at page G offset O.

The 'G' source is the page index used for memory operations via the 'M' source or destination. The 'M' destination stores the source data byte into memory as described. See Memory Addressing for information on implied Page index registers, offset registers and so forth.
