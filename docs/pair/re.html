This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: re
  • Opcode: 200d, C8h
  • Encoding: 1_100_1000b (TRAP_SRC_DST)
  • Pseudo-code: e=r; pc++
  • R:RESULT E:ENABLE

  • Source is the R register.
  • Destination is the E register

The 'E' destination takes the source value as a bit pattern that activates or deactivates connected communication devices. See Device Communication for how this works.
