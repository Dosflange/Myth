This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: sb
  • Opcode: 234d, EAh
  • Encoding: 1_110_1010b (TRAP_SRC_DST)
  • Pseudo-code: pc+=sir
  • S:SERIAL B:BRANCH

  • Source is the Serial Input register (SIR).
  • Add source to the Program Counter.

The 'S' source is a deserialised byte received over the serial bus. See Device Communication for how this works. The 'B' destination takes the source value as the offset relative to the current value of the Program Counter. Writing to 'B' triggers an unconditional jump to the address offset that is the sum of the Program Counter and the source byte within the current code page. See In-Page Branching for more on this.
