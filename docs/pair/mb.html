This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: mb
  • Opcode: 154d, 9Ah
  • Encoding: 1_001_1010b (PAIR_SRC_DST)
  • Pseudo-code: pc+=ram[d][o]
  • M:MEMORY B:BRANCH

  • Source is the memory cell at page D offset O.
  • Add source to the Program Counter.

The 'M' source is the value of a memory cell in the D-page. See Page Registers for more information on the three page register (D, L, C). The 'B' destination takes the source value as the offset relative to the current value of the Program Counter. Writing to 'B' triggers an unconditional jump to the address offset that is the sum of the Program Counter and the source byte within the current code page. See In-Page Branching for more on this.
