This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: mj
  • Opcode: 155d, 9Bh
  • Encoding: 1_001_1011b (PAIR_SRC_DST)
  • Pseudo-code: pc=ram[d][o]
  • M:MEMORY J:JUMP

  • Source is the memory cell at page D offset O.
  • Set the Program Counter to source.

The 'M' source is the value of a memory cell in the D-page. See Page Registers for more information on the three page register (D, L, C). The 'J' destination takes the source value as the target offset of an unconditional jump to within the current code page. See In-Page Branching for more on this.
