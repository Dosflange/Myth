This page describes the following instruction:

  • Group: PAIR
  • Mnemonic: iw
  • Opcode: 220d, DCh
  • Encoding: 1_101_1100b (PAIR_SRC_DST)
  • Pseudo-code: if i!=0 then pc=i else pc++; i--
  • I:INNER W:WHILE

  • Source is the I register.
  • If I is NOT zero, set the Program Counter to source. Then in either case decrement I.

The 'W' destination takes the source value as the target offset of a conditional jump (I != 0) to within the current code page. See In-Page Branching for more on this.
