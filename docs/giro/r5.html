This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: r5
  • Opcode: 109d, 6Dh
  • Encoding: 01_10_1_101b (DIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: L5 = r; pc++

Puts R into local memory at offset 0xFD.
