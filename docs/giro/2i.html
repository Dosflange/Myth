  • Group: GIRO
  • Mnemonic: 2i
  • Opcode: 82d, 52h
  • Encoding: 01_01_0_010b (GIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: i=L2; pc++

Gets I from local memory at offset 0xFA.
