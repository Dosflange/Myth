This page describes the following instruction:

  • Group: GIRO
  • Mnemonic: i5
  • Opcode: 93d, 5Dh
  • Encoding: 01_01_1_101b (GIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: L5 = i; pc++

Puts I into local memory at offset 0xFD.
