This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: d4
  • Opcode: 76d, 4Ch
  • Encoding: 01_00_1_100b (DIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: L4 = d; pc++

Puts D into local memory at offset 0xFC.
