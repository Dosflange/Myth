This page describes the following instruction:

  • Group: DIRO
  • Mnemonic: i2
  • Opcode: 90d, 5Ah
  • Encoding: 01_01_1_010b (DIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: L2 = i; pc++

Puts I into local memory at offset 0xFA.
