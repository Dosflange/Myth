This page describes the following instruction:

  • Group: GIRO
  • Mnemonic: 0g
  • Opcode: 64d, 40h
  • Encoding: 01_00_0_000b (GIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: g=L0; pc++

Gets G from local memory at offset 0xF8.
