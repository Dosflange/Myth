This page describes the following instruction:

  • Group: ALU
  • Mnemonic: SLO
  • Opcode: 21d, 15h
  • Encoding: 0001_0101b (ALU_OPERATION)
  • Pseudo-code: r=o<<1; pc++

This instruction multiplies register O by 2 and stores the result in R.
