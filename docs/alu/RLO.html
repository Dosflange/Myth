This page describes the following instruction:

  • Group: ALU
  • Mnemonic: RLO
  • Opcode: 29d, 1Dh
  • Encoding: 0001_1101b (ALU_OPERATION)
  • Pseudo-code: if r

This instruction sets R to 255 if it is less than O. If R is greater than or equal to O, register R is set to zero.
