This page describes the following instruction:

  • Group: ALU
  • Mnemonic: IDO
  • Opcode: 17d, 11h
  • Encoding: 0001_0001b (ALU_OPERATION)
  • Pseudo-code: r=o; pc++

This instruction copies the value of O into R.
