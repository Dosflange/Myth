This page describes the following instruction:

  • Group: ALU
  • Mnemonic: OCR
  • Opcode: 18d, 12h
  • Encoding: 0001_0010b (ALU_OPERATION)
  • Pseudo-code: r=~r; pc++

This instruction stores the ones' complement (bitwise inversion) of R into R.
