This page describes the following instruction:

  • Group: SYS
  • Mnemonic: SCL
  • Opcode: 3d, 03h
  • Encoding: 00000_011b (SYS_INDEX)
  • Pseudo-code: sclk=0; pc++

This instruction sets the serial clock line to 'LOW'. See Device Communication for details.
