This page describes the following instruction:

  • Group: SYS
  • Mnemonic: RET
  • Opcode: 5d, 05h
  • Encoding: 00000_101b (SYS_INDEX)
  • Pseudo-code: c=L7; pc=i; l++;

This instruction undoes the most recent TRAP, CALL or IRQ. See Subroutines for details.
