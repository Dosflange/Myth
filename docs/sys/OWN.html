This page describes the following instruction:

  • Group: SYS
  • Mnemonic: OWN
  • Opcode: 7d, 07h
  • Encoding: 00000_111b (SYS_INDEX)
  • Encoding: 00001_111b (FIX_ADDEND)
  • Pseudo-code: L7=co; pc++

This instruction saves the current code page index into L7. See Subroutines for details.
