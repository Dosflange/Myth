This page describes the following instruction:

  • Group: SYS
  • Mnemonic: SCH
  • Opcode: 4d, 04h
  • Encoding: 00000_100b (SYS_INDEX)
  • Encoding: 00001_100b (FIX_ADDEND)
  • Pseudo-code: sclk=1; pc++

This instruction sets the serial clock line to 'HIGH'. See Device Communication for details.
